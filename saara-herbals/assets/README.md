# Assets

This folder contains all supporting visual and reference assets used across the Saara Herbals project.

## Contents

- Raw images and ingredient visuals  
- Background textures and design elements  
- Reference materials used for product and brand design  

## Purpose

To maintain a centralized collection of reusable visual assets for consistent design and branding.

## Notes

- Assets should be organized and labeled clearly  
- Only high-quality and relevant files should be stored  
- Avoid duplicates and unused files
